Hello, I'm Cosmin, specializing in Web Application Development

You can try one here

I've always been fascinated by algorithms.

By learning JavaScript, the web has now become an endless playground that lets me show my creativity in various ways.

What I want to demonstrate with this program are my skills and abilities of using well established web technologies, as well as new cutting edge tools.

I have implemented some features, such as complementary color contrast (mix-blend-mode), that are not fully supported by all web browsers.

They were used to highlight different possibilities. Fallback for unsupported browsers coming soon.

Now to the app. How does it work?

The user chooses 6 numbers, then the program generates 6 random numbers as well.

The numbers that match are marked by a different color.

Does real randomness exists, or everything follow a pattern?

I'll try to find out by adding statistics to the project.

Feel free to comment the idea and to propose changes and improvements.
